## 🌈 未來 3.0 願景：AI 智能與社群互動 (開發建議)

### 1. 🤖 AI 語音診斷系統 (推薦首選)
- **語音輸入辨識**：利用 Web Speech API 的 `SpeechRecognition` 功能，讓孩子對著麥克風唸出注音，AI 判斷發音是否精準。
- **糾音小老師**：若發音不準，由語音合成引導：「再試一次喔！這次舌頭要稍微往上翹。」

### 2. 🌩️ Firebase 雲端生態系 (功能躍遷)
- **全球即時排行榜**：將分數上傳 Firestore，讓孩童能看見「全台灣前 100 名」的成就。
- **雲端題庫市集**：家長可以將自訂的精編題庫「公開分享」到雲端，其他老師只要輸入代碼就能一鍵下載。

### 3. 🏫 課堂即時互動系統 (Kahoot 模式)
- **多人競賽房**：由老師端開啟房間，學生端 PWA 掃碼進入，進行全班同步的注音搶對大賽。
- **答題統計圖**：結束後即時產生成績分佈，讓老師知道哪些注音是全班共同的弱點。

### 4. 🎨 沉浸式視覺 3.0
- **動態粒子背景牆**：為每個主題加入更精緻的 Canvas 背景（如：水果關會有落下的微縮水果）。
- **3D 吉祥物動畫**：引入 Three.js 或自繪向量動畫，讓原本靜態的 icon 活過來與孩子互動。
