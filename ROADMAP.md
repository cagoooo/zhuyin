### 1. PWA 離線化（最優先推薦！）
- [ ] **Service Worker 實作**：讓整個遊戲可在無網路環境（飛機、地下室、餐廳）下完整執行。
- [ ] **Web App Manifest**：新增 `manifest.json`，讓家長能將遊戲安裝到手機/iPad 桌面，無瀏覽器 UI 干擾，完全 App 化體驗。
- [ ] **離線圖片快取策略**：所有本地 `./assets/images/*.png` 優先存入 Cache API；CDN 圖片則採用「先快取，更新後才替換」的 stale-while-revalidate 策略。

### 2. AI 生成式教學功能
- [ ] **Gemini API 出題（實驗性）**：在設定頁面提供「AI 出題模式」，由 Gemini Flash 生成適合程度的生詞，再自動查詢注音。
- [ ] **錯題智能復習**：遊玩結束後，AI 自動分析最常錯的注音並在下次出題時優先出現（間隔重複演算法 Spaced Repetition）。
- [ ] **發音評分（Speech-to-Text）**：孩童說出注音，遊戲用 Web Speech Recognition API 比對輸入，訓練「聽說讀寫」四技能。

### 3. 自訂題庫系統（家長功能）
- [ ] **家長後台頁面**：提供簡單的管理介面，讓爸媽輸入「字→注音」並新增至自訂關卡。
- [ ] **QR 碼分享題庫**：將自訂題庫編碼成 Base64 URL，生成 QR 碼讓老師分享給全班掃描匯入。

## 🔴 長期目標（3個月以上）

### 1. Firebase 雲端帳號與多裝置同步
- [ ] **Firebase Authentication**：Google 信箱一鍵登入，讓孩童的進度在家裡的 iPad 與學校的電腦都能同步延續。
- [ ] **Firestore 學習數據庫**：儲存「每個符號的錯誤次數」與「最長連擊紀錄」，建立個人化的學習輪廓 (Learning Profile)。
- [ ] **家長學習報表儀表板**：使用 Chart.js 視覺化「最常出錯的注音符號TOP5」、「各主題完成率」等數據，讓家長掌握孩子學習狀況。

### 2. 社群與班級功能
- [ ] **即時全球排行榜**：透過 Firestore 即時資料庫，將通關時間/Combo 最高分上傳，建立「注音大師排行榜」。
- [ ] **班級房間系統**：老師建立房間，輸入學生姓名，學生掃 QR 碼進入，老師可即時看到全班答題狀況。
- [ ] **完成徽章分享**：通關成就可生成一張「精美獎狀圖片」，讓孩童分享給爸媽或貼到社群媒體打卡。

### 3. 遊戲模式大擴展
- [ ] **聽音輸入模式（聽說訓練）**：不顯示漢字，只播放 TTS 語音，孩童直接拼出注音，訓練純聽力辨音。
- [ ] **故事闖關模式（Story Mode）**：將10個主題串連成一個連貫的冒險故事，每個關卡有專屬的背景故事對話，讓學習有劇情動力。
- [ ] **雙人對戰模式（Local Multiplayer）**：兩位孩童使用同一台裝置，互相比賽搶先按對正確注音，增加競技樂趣。

### 5. 終極教育 AI 整合
- [ ] **Gemini 記憶助記 (Mnemonics)**：當孩童在某個注音卡關時，點擊 AI 圖示由 Gemini 生成口訣，例如「ㄅ像收音機正在廣『播』」。
