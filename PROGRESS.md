## 📌 v2.2.0 已完成功能

### 🎊 採花特效升級 (v2.2.0)
- [x] **60顆多形狀彩帶**：從方塊升級為「圓形、星星、心形、方塊」4種隨機形狀，每顆均有獨立的亮彩顏色（8色彩盤）。
- [x] **物理飄落旋轉**：每顆彩帶隨機產生不同大小(8-20px)、落下速度(1.2-2.4s)與旋轉圈數(720°)，模擬真實物理感。
- [x] **全螢幕視窗覆蓋**：改為 `position: fixed`，直接從螢幕頂端落下至底部，不再侷限於互動區域的小框框。
- [x] **自動清除機制**：每顆彩帶在 3 秒後自動從 DOM 移除，不影響效能。

### 🖼️ 缺圖 404 修正 (v2.2.0)
- [x] **水果關卡**：修正 `橘(mandarin)` 與 `檬(lemon)` 圖片缺失，改為 Icons8 CDN 連結。
- [x] **家人關卡**：修正 `弟(baby-boy)`、`妹(baby-girl)`、`爺(grandfather)` 缺失圖，改為 Icons8 CDN 連結。
- [x] **學校關卡**：修正 `桌(desk)`、`椅(chair)` 缺失圖，改為 Icons8 CDN 連結。

### 🎨 按鈕可讀性強化 (v2.2.0)
- [x] **提示鍵（黃底）**：將文字色強制設為深棕 `#451A03`，移除繼承的白色文字陰影。
- [x] **清除鍵（白底）**：背景改為亮白 `#F1F5F9`，文字改為深石板色 `#1E293B`，大幅提升對比度。
- [x] **送出作答（綠底）**：保持白字，加上立體下壓陰影 `0 5px 0 #15803D`，強化實體按鈕感。

## 📌 v2.1.0 已完成功能

### 🌈 繽紛 UI 與 RWD 細節優化 (v2.1.0)
- [x] **關卡彩虹化**：解鎖關卡按鈕依照 `nth-child` 自動輪替套用 5 種粉彩色系 (Peach, Mint, Sky, Banana, Lilac)，結合 CSS Grid 實作彈性的 RWD 佈局。
- [x] **RWD 關卡按鈕修修**：愛心與文字重疊問題修正，改為 `justify-content: space-evenly` Flex 排版，並用 `clamp()` 響應式縮放各元素。
- [x] **黏土互動泡泡框**：於遊戲核心區域新增一個凸起的白底黏土卡片，使注音題目與答題區更聚焦，並將發音喇叭鈕改為懸浮且帶有呼吸光暈。
- [x] **全彩虛擬鍵盤分類**：為「聲母」、「介音」、「韻母」與「聲調」設定不同的粉彩背景色，幫助孩童利用視覺顏色記憶按鍵區域；同時放大下方系統控制鈕並套用高對比色彩避免誤觸。
