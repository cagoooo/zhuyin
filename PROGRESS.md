## 📌 目前已完成功能 (v2.0.0)

### 🎨 全域 UI/UX 大改版與音效升級 (v2.0.0)
- [x] **Claymorphism 設計系統導入**：捨棄扁平化設計，全面換成 AI 推薦的立體黏土風格 (Claymorphism)，包含雙層陰影、圓潤邊框與高飽和彩盤 (`#4F46E5`, `#F97316`)，大幅提升兒童遊玩時的視覺吸引力。

- [x] **全新互動音效系統**：以 `downloadNewSounds.js` 下載並套用更富童趣的版權自由音檔，替換原本的背景音樂與點擊/狀態回饋音，並加入了 404 Mock 檔案的 Fallback 防呆機制。

## 📌 基礎功能完成清單 (v1.1.0)

### 🚀 新增與優化 (v1.1.0)
- [x] **圖示資源本地化**：透過 Node.js 腳本將原本依賴的外部 CDN (icons8) 資源全數轉移至本地 `/assets/images/`，提升載入速度並防止圖床失效破圖。

- [x] **虛擬鍵盤優化**：調整 CSS 讓按鈕有了按壓的縮放回饋 `transform: scale(0.95)` 與陰影變化，增加手機端觸控手感。
- [x] **GitHub 自動部署準備**：加入 `.gitignore` 且撰寫 `.github/workflows/deploy.yml`，為未來串接 GitHub Pages 的 CI/CD 自動化做好準備。

## 📌 基礎功能完成清單 (v1.0.0)

### 1. 遊戲核心機制
- [x] **關卡解鎖系統**：實作漸進式闖關機制，通過一關才能挑戰下一關。
- [x] **隨機題庫系統**：每關從預設題庫中隨機挑選 5 題，增加遊戲耐玩度。
- [x] **注音輸入鍵盤**：客製化虛擬鍵盤，包含聲母、韻母、聲調，支援組合輸入。
- [x] **答案比對與回饋**：即時比對玩家輸入的注音與正確答案，提供視覺化回饋。

### 2. 學習體驗提升
- [x] **主題式題庫**：內建 6 大主題關卡 (水果🍎、動物🐶、交通🚗、身體✋、家人👨‍👩‍👧‍👦、學校🏫)。
